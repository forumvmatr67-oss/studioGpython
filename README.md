# ⛏️ Minecraft Bot (Python)

[![Python](https://img.shields.io/badge/Python-3.9%2B-blue)]()

Бот для управления Minecraft-сервером через RCON.

## 🚀 Быстрый старт
1. Установите зависимости: `pip install -r requirements.txt`
2. Скопируйте `.env.example` → `.env` и заполните данные сервера.
3. Запустите: `python src/bot/main.py`

## 📄 Команды
- `!list` — список игроков
- `!say <текст>` — отправить сообщение в чат
- `!backup` — создать бэкап (требуется плагин)
- `!help` — справка
