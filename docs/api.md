# API бота (внутреннее)

## RCONClient
- `connect()` — установка соединения и аутентификация
- `send_command(cmd)` — отправка команды, возвращает ответ сервера

## Команды бота (через stdin)
- `!list` — показать онлайн игроков и сохранить кэш
- `!say <текст>` — отправить сообщение в чат
- `!backup` — запустить бэкап (требуется плагин)
- `!help` — справка
